# 0.1.0 — 2026-05-24

## Added

- **`census_list_datasets`** — browse available datasets (ACS5, ACS1, Population Estimates, Decennial) with vintage years
- **`census_list_geographies`** — list geography levels and parent requirements for a dataset+year
- **`census_search_variables`** — keyword search across 28K+ variable labels and concepts with relevance ranking
- **`census_get_variable`** — fetch full metadata for one or more variable codes including MOE sibling references
- **`census_resolve_geography`** — convert place names and addresses to Census FIPS identifiers via TIGERweb and Census Geocoder
- **`census_query_data`** — query ACS/decennial data for variables at a specific geography with suppression code resolution
- **`census_compare_geographies`** — rank and compare variables across multiple geographies with sorting and truncation
- **`CensusApiService`** — Census Data API client with retry/backoff and suppression code mapping
- **`VariableCacheService`** — in-process variables.json cache (default 24h TTL) with keyword search
- **`GeographyService`** — TIGERweb named-place resolution and Census Geocoder address-to-tract resolution
- `CENSUS_API_KEY`, `CENSUS_DEFAULT_YEAR`, `CENSUS_VARIABLE_CACHE_TTL_HOURS` env vars

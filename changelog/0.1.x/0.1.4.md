# 0.1.4 — 2026-05-24

## Changed

- **Build scripts**: all `tsx` invocations replaced with `bun run` — `build`, `rebuild`, `clean`, `devcheck`, `tree`, `list-skills`, `lint:mcp`, `lint:packaging`, `changelog:build`, `changelog:check`
- **`test` script**: switched from `vitest run` to `bunx vitest run` for explicit runner resolution
- **`start:stdio` / `start:http`**: switched from `node dist/index.js` to `bun ./dist/index.js`
- **`start` script**: removed (superseded by `start:stdio` and `start:http`)
- **`engines.bun`**: minimum version raised from `>=1.3.0` to `>=1.3.2`

## Removed

- **`tsx`** devDependency dropped — no longer needed; all scripts run natively under Bun

## Added

- **`funding`** block in `package.json` — GitHub Sponsors + Buy Me a Coffee links
